# Rejestracja Rodzic-Dziecko

Wtyczka do systemu WordPress umożliwiająca rejestrację użytkowników z zachowaniem relacji rodzinnych. System pozwala rejestrować konta jako **Rodzic** (generując unikalny token powiązania) oraz jako **Dziecko** (wymagając podania tokenu rodzica), automatycznie mapując relacje w dedykowanych tabelach bazy danych.

## Uruchomienie w trybie dev
Wejdź do folderu ze wtyczką i potem wpisz:

``
npm install
``

``
npm run start
``
